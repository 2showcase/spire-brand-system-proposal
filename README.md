## 📌 Executive Summary
This repository contains the live implementation of a brand alignment system developed specifically for the **Spire Tea Head Office**. 

The goal of this project is to address the core challenges of a global B2B supplier: **reducing visual variation** and **improving communication consistency** across digital and physical touchpoints without the need for a full website restructure.

## 🛠 Project Scope & Deliverables
This system was built to be modular and "implementation-ready," focusing on three primary pillars:

### 01. Visual Identity Architecture
* **Typography:** Implementation of *Libre Baskerville* (Heritage/Trust) and *Inter* (Digital Clarity).
* **Color Logic:** A botanical-industrial palette (Forest, Charcoal, Copper) optimized for B2B credibility.
* **The Grid:** An 8px baseline grid system to ensure spacing consistency.

### 02. Web Brand Alignment (Non-Destructive)
* Standardized UI components (Buttons, Headers, Footers).
* Defined vertical rhythm and line-height (150%) for improved readability.
* Maximum container widths (680px for text) to prevent layout fragmentation on large displays.

### 03. Operational Correspondence
* Copy-paste ready HTML email signatures.
* Structured Product Information Sheet templates designed for traceability and sales clarity.

---

## 📐 Design Philosophy: "System over Visuals"
For a global supplier like Spire, isolated "pretty" designs create friction. I prioritize **scalable documentation** and **repeatable systems**. 

> "Consistency in branding is a proxy for consistency in the supply chain. If the documentation is precise, the client trusts the product."

---

## 🚀 Technical Implementation
* **Core:** Clean HTML5 / CSS3 (No heavy frameworks for maximum load speed).
* **Responsiveness:** Mobile-first architecture for logistics teams in the field.
* **Accessibility:** High-contrast ratios and web-safe font fallbacks.

## 📁 Repository Structure
* `index.html`: The live proposal and brand overview.
* `assets/`: PDF documentation and brand overview screenshots.
* `styles/`: Modular CSS defining the brand's visual rules.
